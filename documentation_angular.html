About Template
With the Agular HTML project here we have added angular start up project most of the jQuery plugin demo we have included. Her pages and demo are just for navigation and element working with the angular presentation. We have not used or craeted any data services so you can start with scratch.

We also created themes with diffrent css you can swich other style theme with saperate css so we can reduce page load while writing and overiting css properties. This template also includes SCSS files which can be easily modified as per you need. We also customized variables properties of bootstrape scss and made it with 16 column grid structure. Grid structure can be changed by change in variables file of bootstrap scss. We have made our template based on 16 column stucture to make it fit with our UI to provide better responsive space in smaller view port sizes.

This is all about this template. Hope you like our template and dont forget to share your experience.

CSS
As you know css are used for applying formating and theming to ements of htmls, We have created adminux css with the help of scss and included into css folder. We have choosen scss for its flexibility and scalibility using varibles. You can easily chnage few colors to make your own templete theme. You can put your own brand colors and create your theme by adminux.scss. We have used sout app scss compiler GUI tool. We have also predefined css with different color scheme. To change the stylesheet you just need to replace dark_blue_adminux.css to your favorite theme from.angular-cli.json in root folder of angular. Diffrent css will reduce the code of css in single css file theming.

"assets/scss/dark_blue_adminux.css"
Other css those can be switch are: 
"assets/scss/dark_purple_adminux.css"
"assets/scss/dark_red_adminux.css"
"assets/scss/dark_green_adminux.css"
"assets/scss/dark_grey_adminux.css"
"assets/scss/dark_brown_adminux.css"
".assetscsss/light_adminux.css"
